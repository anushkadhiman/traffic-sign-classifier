# traffic-sign-classifier

Used Convolutional neural networks (CNN) for road sign recognition. 

Dataset contains 978 ppm format images of road sign. Train on 782 samples, validate on 196 samples.

# Library Used
- Keras
- Sklearn
- pandas
- numpy
